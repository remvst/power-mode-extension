# Power Mode

Power Mode browser extension.

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

This watches for changes and rebuilds automatically.

## Build

```bash
npm run build
```

Output is in `dist/`. Load this folder as an unpacked extension in your browser.

## Scripts

- `npm run dev` - Watch mode for development
- `npm run build` - Production build
- `npm run format` - Format code with Prettier
- `npm run typecheck` - Run TypeScript type checking
- `npm test` - Run format check and typecheck
